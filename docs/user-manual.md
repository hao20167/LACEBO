# LACEBO User Manual

> Version: 1.0
> Project: LACEBO – Roleplay Community Platform

---

# 1. Introduction

## 1.1 About LACEBO

LACEBO is a Roleplay Community platform that allows users to create, join, and develop fictional worlds through events, story-driven posts, and community interactions.

### Key Features

- User account management
- Create and join Worlds
- Event management
- Story-based posting system
- Comments and reactions
- Credits and Leaderboard system
- Community moderation tools

---

# 2. Getting Started

## 2.1 Register an Account

1. Navigate to the **Register** page.
2. Fill in the following information:
   - Display Name
   - Username
   - Email
   - Password
3. Click **Register**.
4. Your account will be created successfully.

---

## 2.2 Login

1. Navigate to the **Login** page.
2. Enter:
   - Username
   - Password
3. Click **Login**.

After successful login, you will be redirected to the Home page.

---

## 2.3 Logout

1. Click the account menu.
2. Select **Logout**.

---

# 5. Worlds

## 5.1 What is a World?

A World is an independent roleplay community containing:

- Members
- Storyline
- Events
- Posts
- Leaderboard

---

## 5.2 Search Worlds

1. Navigate to **Worlds**.
2. Enter keywords in the search box.
3. Matching Worlds will be displayed automatically.

---

## 5.3 Create a World

1. Click **Create World**.
2. Enter:
   - Title
   - Description
3. Click **Create**.

The creator automatically becomes the World Developer.

---

## 5.4 Join a World

1. Open the World Detail page.
2. Click **Join World**.
3. Submit a join request.

The request will be reviewed by the World Developer.

---

## 5.5 My Worlds

Navigate to **My Worlds** to view:

- World Name
- Membership Status
- Role
- Credits

---

# 7. Events

## 7.1 What is an Event?

An Event is a storyline activity within a World that drives community interaction and roleplay progression.

---

## 7.2 View Event

The Event Detail page contains:

- Event information
- Related posts
- Comments
- Reactions

---

## 7.3 Event Statuses

| Status | Description |
|----------|-------------|
| Proposed | Waiting for approval |
| Approved | Approved by moderators |
| Open | Currently active |
| Closed | Finished |

# 11. Credits System

## 11.1 What are Credits?

Credits measure a member's contribution within a World.

---

## 11.2 Earning Credits

| Activity | Credits |
|-----------|----------|
| Approved Post | +10 |
| Comment | +2 |
| Received Like | +1 |
| Lost Like | -1 |

---

## 11.3 Leaderboard Ranking

Credits are used to:

- Calculate rankings
- Display Leaderboards
- Measure contributions

# 16. FAQ

## Why can't I join a World?

Possible reasons:

- You are not logged in.
- Your request is pending approval.
- Your request was rejected.

---

## Why are my Credits not increasing?

Check whether:

- Your post has been approved.
- Your comment was successfully submitted.
- Your received likes have been counted.

---

## I forgot my password.

Contact the system administrator or use the password recovery feature if available.
