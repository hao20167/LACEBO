\section{Tổng quan dự án}
% ==========================================

LACEBO (lấy cảm hứng từ \textit{Placebo} -- giả dược) là một nền tảng mạng xã hội nhập vai (roleplay), nơi người dùng có thể tạo và tham gia các thế giới ảo dưới dạng đơn giản. Dự án được xây dựng theo kiến trúc \textbf{Client--Server} với:

\begin{itemize}[leftmargin=2cm]
  \item \textbf{Backend:} Node.js + Express.js + SQLite (better-sqlite3)
  \item \textbf{Frontend:} React 18 + Vite + TailwindCSS
  \item \textbf{Authentication:} JWT (JSON Web Token)
\end{itemize}

\subsection{Database Schema (ERD tóm tắt)}

Hệ thống sử dụng 7 bảng chính:

\begin{enumerate}[leftmargin=2cm]
  \item \textbf{users} -- Thông tin người dùng (username, email, password hash, display\_name)
  \item \textbf{worlds} -- Các thế giới roleplay (title, description, is\_public)
  \item \textbf{world\_members} -- Quan hệ N-N giữa user và world (role: dev/player, status, credits)
  \item \textbf{events} -- Sự kiện trong world (big/small, status: proposed/approved/open/closed)
  \item \textbf{posts} -- Bài viết trong event (content, status: pending/approved)
  \item \textbf{comments} -- Bình luận trên post
  \item \textbf{likes} -- Lượt thích (post hoặc comment)
  \item \textbf{announcements} -- Thông báo của Dev cho world
\end{enumerate}
